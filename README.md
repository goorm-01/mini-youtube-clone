# mini-youtube-clone

## 개요

본 프로젝트는 HTML과 CSS를 활용하여 유튜브 웹 사이트의 UI 구조를 분석하고 이를 기반으로 목업을 구현하는 미니 프로젝트입니다.
화면 구성과 레이아웃 설계, 시멘틱한 DOM 구조 이해를 목표로 합니다.
팀 단위로 진행되며 각 구성원은 유튜브 사이트의 특정 페이지를 맡아 개발한 뒤 이를 통합합니다.

## 프로젝트 목표

1. 유튜브 UI 구조 분석 및 화면 구성 요소 분해
2. HTML을 활용한 시맨틱 구조 설계
3. CSS를 활용한 기본 스타일링 및 레이아웃 구성
5. DOM 트리 구조의 이해
6. Header / Thumbnail 등 핵심 UI 요소 구현
7. 화면 설계서 및 결과 문서 작성

## 페이지별 역할 분담

각 팀원은 유튜브 페이지 단위로 역할을 분담하여 개발을 진행합니다.

|페이지|담당영역|
|--|--|
|메인 페이지|영상 카드 목록, 전체 레이아웃|
|영상 페이지|영상 플레이 영역 및 상세 정보 UI, 추천 영상 레이아웃|
|채널 페이지|채널 정보, 채널 영상 리스트 목록|
|쇼츠 페이지|쇼츠 카드 레이아웃, 댓글 리스트 목록|
|검색 페이지|검색 결과 목록 UI|

## 개발 방식

- 각 페이지는 개별 브랜치에서 개발합니다.
- 공통 컴포넌트는 별도로 관리합니다.
- 개발 완료 후 release 브랜치에서 QA를 진행합니다.
